# **Snake_like_a_pro**
Игра на консоли "Змейка" на Python

## **Руководство к запуску**
* Открыть терминал
* Устанавливаем PyGame
	необходимо ввести следующую строку в терминал:
	```pip install pygame```
* Запуск необходимого файла
	вводим в терминале:
	```python main.py```
## **Управление змейкой**
* Повороты змейки:
	* &uArr; - вверх
	* &dArr; - вниз
	* &lArr; - влево
	* &rArr; - вправо
* Выход из игры - Q-quit
* Начало игры заново - C-play again
